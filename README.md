# LazerAssaultv2

A simple, terminal-based lazer obstacle game created using Potato Engine 2.0.4-beta

This game was developed on Windows using MSVC

## Objective
Live as long as you can and get a high score! Lazers activate periodically and can damage you. Be careful, because you only have 100 health!

## Controls
Use `WASD` to move player  
Press `Escape` to quit the game at any time
